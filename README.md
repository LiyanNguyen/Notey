# Notey
A fullstack web application that implements CRUD operations.

You can create notes, assign colors and ratings to it, edit, delete, update, sort, filter the notes to your liking.


### Technologies Used
- Vite
- React
- Tailwind
- Recoil
- Tanstack Query
- Supabase
